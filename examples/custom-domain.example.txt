Custom domain example for ULTRA 1.0 (GitHub Pages)
==================================================

1) This project is configured for the custom domain:
     cam-ultra-planner-2026.app
   Register that name at your registrar if you don’t own it yet.

2) DNS — apex (.app root hostname):
   Use GitHub’s A + AAAA records for an apex domain (see GitHub Pages docs), or an
   ALIAS/ANAME to camdigitydog-ux.github.io if your DNS provider supports it.
   (A plain CNAME at the zone apex is often invalid; subdomains like www CAN use CNAME.)

3) In this repo, the root file CNAME (no extension) is one line only:
     cam-ultra-planner-2026.app

4) GitHub: Repository → Settings → Pages → Custom domain → same hostname → Save.
   Turn on “Enforce HTTPS” when GitHub allows it.

Reserved names for documentation (do NOT register — you can’t use for real sites):
  example.com, example.net, example.org (see RFC 2606)
